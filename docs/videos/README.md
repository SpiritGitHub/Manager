# Vidéos / démonstrations

Déposez ici les vidéos de démonstration (ex: MP4) et référencez-les depuis le README.

## Recommandation (GitHub README)

- Les **images** et **GIF** s'affichent directement dans le README via `![](...)`.
- Les **vidéos MP4** dans un dépôt sont généralement **cliquables** (lecture dans le navigateur selon le contexte), mais l'"embed" direct dans un README n'est pas garanti.
- Pour une vidéo toujours lisible, utilisez un lien **YouTube** (ou équivalent) + une vignette image dans `docs/screenshots/`.

## Convention de nommage (recommandée)

- `demo.mp4`
- `tour-rapide.mp4`

Astuce : gardez des noms simples et stables pour éviter de casser les liens.
